# react-native-todo

1. cd react-native-todo && npm install
2. npx expo start

Réalisation d'un TodoList avec react-native

- Créer une nouvelle tâche
- Supression d'une tâche
- Sauvegarde de la liste des tâches avec AsyncStorage
- Filtre sur les tâches "all | inProgress | completed"
- Gestion de modals
